Denizen: Helpful Citizens for Bukkit!

**Version 1.0.1**: Compatible with Spigot 1.8.8, 1.9.4, 1.10.2, and 1.11.2!

**Download (Developmental builds)**: http://ci.citizensnpcs.co/job/Denizen/
